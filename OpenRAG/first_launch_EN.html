Setting Up the OpenRAG Application

Once the application code has been downloaded and opened in your code editor, navigate to the open_rag folder and run the following commands in your terminal to install all the necessary components for the application to work:

Navigate to the Docker Directory

cd docker (This opens the docker directory)

Launch the Appropriate Docker Compose File

sudo docker compose -f [DOCKER FILE NAME] up -d

Available Docker Compose Files:

  • docker-compose-all.yml: launches all services (Elasticsearch, the frontend, Ollama, and VLLM)
  • docker-compose-api.yml: launches Elasticsearch and the frontend only; OpenAI and Mistral will be usable
  • docker-compose-ollama.yml: launches only Elasticsearch, the frontend, and Ollama; requires a GPU
  • docker-compose-vllm.yml: launches Elasticsearch, the frontend, and VLLM; requires a GPU

To choose which Docker Compose file to run, you need to decide which LLM provider you want to use for your application (OpenAI, Mistral AI, Ollama, or VLLM).

Access the Application

Once the Docker containers are up and running (this may take a few minutes), open your web browser and go to the following URL: http://localhost:8506/

Once the App Has Loaded, You Can:

  • Set configuration to your needs
  • Add your own data
  • Index RAG method with your own data:
  • Chat with each method
  • Upload your own set of queries (with the associated answers) and perform a benchmark on a chosen RAG method using your own data
